# Chhatrapati Digital

Chhatrapati Digital is a premium digital agency platform offering services like Graphic Design, Event Photography, Premium Signage, and Web Development. This project includes a client-facing website and an administrative dashboard for managing inquiries, client tickets, invoices, and website content (CMS).

## Features

*   **Client Portal**: A beautiful landing page showcasing services, portfolio, and pricing.
*   **Admin Dashboard**: Manage incoming client inquiries, support tickets, and invoices.
*   **Content Management System (CMS)**: Update pricing packages, services, portfolio items, and the "Our Story" section directly from the admin panel.
*   **Ticketing System**: Track and resolve client support requests.

## Technologies Used

*   **Backend:** Python with Flask
*   **Database:** SQLite
*   **Frontend:** HTML5, CSS3, JavaScript
*   **Styling:** Vanilla CSS with FontAwesome Icons

## Setup Instructions

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/kalmeshwargurav1028-sys/Chhatrapati-Digital.git
    cd Chhatrapati-Digital
    ```

2.  **Install dependencies**:
    Make sure you have Python installed. You can install the necessary packages using:
    ```bash
    pip install Flask
    ```
    *(If a `requirements.txt` is present, run `pip install -r requirements.txt`)*

3.  **Run the application**:
    Start the Flask development server by running:
    ```bash
    python app.py
    ```

4.  **Access the web app**:
    *   **Main Website:** `http://127.0.0.1:5000/`
    *   **Admin Dashboard:** `http://127.0.0.1:5000/admin/inquiries`
